# 产品简报：适文（Content Repurposer）

## 执行摘要

适文是国内首个专注于"按平台调性 AI 改写"的内容工具 -- 内容创作领域的"翻译层"。用户输入一篇文章，AI 理解其核心观点后，自动生成适配小红书、微信公众号、知乎、抖音等国内主流平台风格的版本。不是简单的缩写扩写，而是从论点结构、语言风格、互动模式上全面适配目标平台。

国内已有融媒宝、蚁小二解决"发到哪"的问题，有一云AI、壹伴解决"怎么排版"的问题。但"理解内容语义、按平台调性深度改写"这个中间地带仍是空白 -- 现有工具要么原文搬运（格式适配但内容不变），要么 AI 从零生成（质量不可控）。适文切入的正是这个缝隙：让同一篇内容在每个平台都像原生内容。

与直接使用 ChatGPT 等通用 AI 的区别：用户不需要自己写 prompt、不需要懂各平台规则、不需要逐个平台调试 -- 一键出完整的"内容发布包"，开箱即用。

## 问题

一个自媒体运营每天面对 3-5 个内容平台，每个平台有截然不同的内容规范：

- **小红书**：口语化、短句、emoji、体验感、结论前置
- **微信公众号**：长文、深度、论点递进、专业感
- **知乎**：问答逻辑、分析性、数据支撑、理性语气
- **抖音脚本**：口播节奏、悬念开头、短句密集、行动号召

写完一篇公众号文章后，手动改写成其他平台版本平均耗时 20-40 分钟/篇。一天改写 3 个平台就要 1-2 小时。更痛的是，很多创作者只擅长一种平台的写作风格 -- 一个习惯小红书语体的博主，让她写公众号长文比从零写还难。

结果是：大多数创作者只深耕 1 个平台，其余平台要么荒废，要么搬运原文效果很差。

## 解决方案

适文提供"语义级"内容改写。核心流程 4 步完成：

1. **输入内容** -- 粘贴文章全文或输入 URL 自动提取
2. **选择平台** -- 勾选目标平台（小红书/公众号/知乎/抖音脚本）
3. **AI 改写** -- 提取核心论点后，按目标平台的内容结构、语言风格、互动模式重新组织表达；支持长文拆解为系列短内容（知识付费高频场景）
4. **复制使用** -- 每个平台输出完整的"内容包"：改写文案 + 3 个备选标题 + 推荐标签 + 互动引导语 + 平台适配度反馈（帮助用户判断发布时机）

用户可通过"语气强度调节器"微调输出风格（从"极度口语化"到"非常正式"自由滑动），实时预览变化。全程流式输出，改写结果逐字呈现，消除等待焦虑。

## 差异化优势

**与分发工具（融媒宝/蚁小二）的关系：** 互补而非竞争。它们解决"发到哪"，我们解决"怎么写才适合那个平台"。未来通过 API 打通，形成"适文改写 -> 融媒宝分发"的完整链路。

**与 AI 写作工具（有一云/壹伴）的区别：** 它们侧重公众号排版和从零生成内容。我们不做排版，只做已有内容的跨平台改写，做到极致深度。

**与通用 AI（ChatGPT/文心一言）的区别：** 通用 AI 需要用户自己写 prompt、理解平台规则、逐个平台调试。适文把这些专业知识内置为产品能力 -- 按平台预置的专属 prompt 链 + 真实爆文语料 few-shot 示例 + 实时更新的平台规则适配层。用户零学习成本，开箱即用。

## 目标用户

**主力付费用户：**
- **企业新媒体运营**（月付 99-299 元）-- 管理 3-5 个平台，产出压力大，需要品牌调性一致
- **知识付费从业者**（月付 99-199 元）-- 擅长深度内容，需要"降维表达"触达年轻用户；长文拆解为系列短内容是高频刚需
- **MCN 机构**（企业定制）-- 管理上百个账号，每天大量改写需求，付费能力最强

**增长用户：**
- **个人自媒体博主**（免费/月付 29 元）-- 价格敏感但数量庞大，是口碑传播和自然获客的核心群体

## 定价结构

三档定价，覆盖全用户层次：

| 套餐 | 价格 | 主要限制 | 目标用户 |
|---|---|---|---|
| **免费版** | 0 元 | 2 个平台 + 3 次/天 | 体验用户、个人博主 |
| **个人版** | 29 元/月 | 全部平台 + 无限次数 | 个人创作者、知识付费从业者 |
| **团队版** | 199 元/月 | 多成员 + 品牌调性模板 + 批量处理 | 企业新媒体运营、MCN 机构 |

## 成功标准

| 指标 | 目标（上线 3 个月内） | 衡量方式 |
|---|---|---|
| 注册用户 | 5,000+ | 注册数据 |
| 日活跃改写次数 | 1,000+ 次/天 | 产品埋点 |
| 付费转化率 | 免费 -> 付费 5%+ | 转化漏斗分析 |
| 改写可用率 | 用户编辑幅度 <30% 即可发布的比例 > 70% | 用户行为数据（复制后是否回来重新生成） |
| NPS | > 40 | 定期用户调研 |

## 范围

**MVP 包含：**
- 文章输入（粘贴全文 / URL 自动提取正文）
- 4 个目标平台（小红书、微信公众号、知乎、抖音口播脚本）
- AI 语义级改写（风格 + 结构 + 用语全面适配）
- 多维度内容包输出（文案 + 标题 + 标签 + 互动引导语）
- 语气强度微调（滑块控制）
- 历史记录与内容管理
- 流式输出（streaming）体验

**明确不做：**
- 排版 / 图片处理
- 多平台一键发布 / 账号管理
- 定时发布 / 数据分析
- 从零 AI 生成内容（只做改写，不做创作）

**技术约束：**
- 单次改写 API 调用成本需控制在可持续范围内（影响定价模型）
- 平台规则配置层需与 AI 改写逻辑解耦，确保规则变更时快速适配

## 愿景

适文的长期定位是**内容创作领域的"翻译层"** -- 不替代任何平台的创作工具，而是连接所有平台的语义适配基础设施。

发展路径：

1. **V1：改写工具**（当前）-- 单篇内容跨平台改写，Web 端为主，建立用户基础
2. **V1.5：浏览器插件** -- 在内容平台页面内一键改写，降低使用门槛，扩大自然获客
3. **V2：智能助手** -- 个人风格学习、爆文策略融合、品牌调性模板，成为创作者的"多平台内容顾问"
4. **V3：内容基础设施** -- 开放 API 与融媒宝/蚁小二等分发工具打通，成为内容创作链条中"改写"环节的标准组件

**初期获客方向：** 通过内容营销（"小红书文章怎么改成公众号""公众号转知乎技巧"等长尾 SEO 内容）获取精准用户 + 免费版自然口碑传播。
